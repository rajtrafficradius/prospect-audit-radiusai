# Integrated SEO, AEO & GEO Quality Assurance Checklist

This checklist must be used to validate the final deliverables. The goal is to ensure every audit package is professional, consistent, high-quality, and fully integrates the three layers of modern search.

---

## Deliverable-Specific Checks

### All Deliverables (DOCX, XLSX, Slides)

| Check ID | Item | Requirement |
|---|---|---|
| GEN-01 | Branding | TrafficRadius logo is present and correctly placed. |
| GEN-02 | Colors | The correct brand color palette (navy, blue, white) is used. |
| GEN-03 | Copyright | The footer "© Copyright 2026 TrafficRadius, All rights reserved." is present on every page/slide. |
| GEN-04 | Personalization | The prospect's company name is correctly spelled and used throughout. |
| GEN-05 | Tone | The language is professional, positive, and opportunity-focused. |

### DOCX Strategy Document

| Check ID | Item | Requirement |
|---|---|---|
| DOCX-01 | Structure | The document contains all 12 sections as per the updated `strategy_template.md`. |
| DOCX-02 | Charts | All generated charts are embedded correctly and are legible. |
| DOCX-03 | Tables | All tables are correctly formatted with branded headers. |
| DOCX-04 | Data Integrity | All data points and metrics match the source JSON/CSV files. |
| DOCX-05 | **AEO/GEO Integration** | The audit section explicitly details AEO and GEO findings (e.g., schema gaps, AI bot blocks). |
| DOCX-06 | **AEO/GEO Integration** | The strategy section clearly defines separate pillars/actions for SEO, AEO, and GEO. |

### XLSX Workbooks (Keywords & Action Plan)

| Check ID | Item | Requirement |
|---|---|---|
| XLSX-01 | Branding | The TrafficRadius logo is present on each sheet. |
| XLSX-02 | Formatting | Headers are branded, and data is clean and well-organized. |
| XLSX-03 | Keyword Data | The keyword workbook contains all required sheets and columns. |
| XLSX-04 | Action Plan | The action plan workbook is broken down by month and contains all required columns. |
| XLSX-05 | **AEO/GEO Integration** | The keyword workbook includes a column identifying the opportunity type (SEO, AEO, GEO). |
| XLSX-06 | **AEO/GEO Integration** | The action plan includes specific, distinct tasks for SEO, AEO, and GEO initiatives. |

### Slide Deck

| Check ID | Item | Requirement |
|---|---|---|
| SLIDE-01 | Structure | The deck follows the 16-20 slide structure from the updated `slide_outline_template.md`. |
| SLIDE-02 | Visuals | **Every slide contains a custom CSS diagram or data visualization.** There are no bullet-point-only slides. |
| SLIDE-03 | Design | The design is clean, modern, and uses the white/light grey theme. |
| SLIDE-04 | Readability | All text is legible, and charts/diagrams are easy to understand. |
| SLIDE-05 | **AEO/GEO Integration** | The deck includes the educational "Three Layers of Modern Search" slide. |
| SLIDE-06 | **AEO/GEO Integration** | The strategy and action plan slides clearly differentiate between SEO, AEO, and GEO initiatives. |

---

## Content & Strategy Checks

| Check ID | Item | Requirement |
|---|---|---|
| STRAT-01 | No Projections | There are **zero** traffic, revenue, lead, or ranking projections or guarantees. |
| STRAT-02 | Strategic Level | The content remains at a strategic level (the *what* and *why*), avoiding execution-level detail (the *how*). |
| STRAT-03 | Data Source | All data from SEMrush is implicitly or explicitly sourced as such. No data is fabricated. |
| STRAT-04 | Keyword Count | The keyword research contains at least 150 high-relevance opportunities. |
| STRAT-05 | Competitor Count | At least 3 direct competitors have been analyzed. |
| STRAT-06 | **AEO Check** | AEO-specific opportunities (e.g., Featured Snippets, PAA) are clearly identified in the analysis. |
| STRAT-07 | **GEO Check** | GEO-specific audit findings (e.g., AI bot accessibility, entity signals) are included and explained. |
| STRAT-08 | **Integrated Strategy** | The strategy clearly articulates the difference and synergy between SEO, AEO, and GEO, presenting them as an integrated whole. |
